# 🌿 Agro — Fresh Organic Store

Agro is a modern **e-commerce frontend application** built with **Next.js, React, TypeScript, and Tailwind CSS**, designed to deliver a premium shopping experience for fresh and organic products.

---

## 🚀 Live Demo

👉 https://your-vercel-link.vercel.app/

---

## ✨ Features

### 🛍️ Shopping Experience
- Browse products with category & search filter
- Premium product cards with clean UI
- Dynamic product details page
- Add to cart with quantity control (0–10)
- Real-time cart updates

---

### 🛒 Cart System
- Add / remove products
- Increase / decrease quantity
- Direct quantity input
- Auto-remove when quantity = 0
- Cart persistence using localStorage

---

### 💳 Checkout Flow
- User-friendly checkout form
- Auto-filled user details (from profile)
- Validation (phone, pincode, etc.)
- Payment method selection (UPI / Card / COD)
- Order summary with total calculation

---

### 🎉 Order System
- Order success page with custom confetti 🎊
- Stores orders in localStorage
- Order history inside profile page
- Order status:
  - Processing
  - Packed
  - Delivered

---

### 👤 Authentication
- Login / Signup system
- User data stored locally
- Protected checkout flow

---

### 🌐 Multi-language Support
- English 🇬🇧
- Hindi 🇮🇳
- Dynamic UI text switching

---

### 🎨 UI / UX
- Premium glassmorphism design
- Fully responsive layout
- Sticky animated navbar
- Smooth transitions & hover effects
